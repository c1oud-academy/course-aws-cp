Қайырлы күн, құрметті студенттер! Сіздердің барлығыңызды келесі Practice Test сұрағын талдау сессиясында көргеніме қуаныштымын. Practice Test сұрақтары нақты AWS емтихан сұрақтарына жақын сұрақтар болып табылады. Осылайша, біздің платформадағы Practice Test-тер бойынша жаттығып, сондай-ақ барлық Practice Test сұрақтарын талдау арқылы сіз нақты AWS емтиханын сәтті тапсыру мүмкіндігіңізді арттырасыз. Біз талдау үшін ең қиын сұрақтарды таңдаймыз, сондықтан сіз осы сұрақтарға қалай жауап беру керектігін білгеннен кейін, қалғандарына жауап беру қиын болмайды. Сұрақтың мазмұнына ғана емес, сонымен қатар дұрыс жауап табу үшін қандай әрекеттер жасайтынымызға назар аудару керек екенін атап өткім келеді. Сонымен, бастайық. Сұрақ: “Default Network ACL және шығыс трафигі туралы келесі мәлімдемелердің қайсысы дұрыс?” Жарайды, сұраққа жауап беруге тырысайық. Бұл көлемді сұрақтың мысалы. Жауап нұсқалары өте ұзын екенін көруге болады. Бұдан қорықпаңыз, керісінше, мұндай сұрақтар қарапайым, тек егжей-тегжейлі қарау жеткілікті. Осылайша, егер біз A және D нұсқаларына қарасақ, басталуы бірдей. Default NACLs are stateful. B және C жауап нұсқалары да ұқсас. "Default NACLs are stateless". Біз дәрістен Network ACL - stateless, ал security groups - stateful екенін білеміз. Осылайша, бірден А және D нұсқаларын алып тастаймыз. Бізде тек екі нұсқа қалды. 50 де 50. Оқып көрейік. "Default NACLs are stateless and they allow or block all outbound traffic by default", яғни шығыс трафиктің бұғатталуымен немесе рұқсат етілуімен ерекшеленеді. Бұл туралы дәрісте айттық. Егер біз Default Network ACL туралы айтатын болсақ, кіріс және шығыс трафик әдепкі бойынша рұқсат етіледі. Ал егер, біз Custom Network ACL туралы айтсақ, яғни Default Network ACL үстінен қосымша құрылған Network ACL, әдепкі бойынша барлық кіріс және шығыс трафикке тыйым салынады. Осылайша, Default Network ACL және Outbound traffic үшін дұрыс мәлімдеме B нұсқасы болып табылады. Осымен біз Practice Test сұрағына талдауымызды аяқтаймыз. Сіз жаңа нәрсе білдіңіз деп үміттенемін. Келесі белсенділіктерімізде көріскенше!
