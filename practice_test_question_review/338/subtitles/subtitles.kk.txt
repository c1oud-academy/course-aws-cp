Қайырлы күн, құрметті студенттер! 
Сіздердің барлығыңызды келесі Practice Test сұрағын талдау сессиясында көргеніме қуаныштымын.
Practice Test сұрақтары нақты AWS емтихан сұрақтарына жақын сұрақтар болып табылады.
Осылайша, біздің платформадағы Practice Test-тер бойынша жаттығып, сондай-ақ барлық Practice Test сұрақтарын талдау арқылы сіз нақты AWS емтиханын сәтті тапсыру мүмкіндігіңізді арттырасыз.
Біз талдау үшін ең қиын сұрақтарды таңдаймыз, сондықтан сіз осы сұрақтарға қалай жауап беру керектігін білгеннен кейін, қалғандарына жауап беру қиын болмайды.
Сұрақтың мазмұнына ғана емес, сонымен қатар дұрыс жауап табу үшін қандай әрекеттер жасайтынымызға назар аудару керек екенін атап өткім келеді.
Сонымен, бастайық.
Сұрақ: «KMS сервисіне импортталған шифрлау кілті немесе осы сервисте жасалған кілт арасындағы негізгі айырмашылықтар қандай?»
Бұл сұраққа біз екі дұрыс жауапты таңдауымыз керек.
Жарайды, осы сұраққа жауап беруге тырысайық.
Бірінші нұсқа A - You cannot actually import keys.
You can only generate keys in AWS KMS.
Бұл мәлімдеме жалған, сондықтан оны дұрыс емес деп белгілейміз.
KMS ішінде біз импортталған кілттермен жұмыс істей аламыз, сондай-ақ оларды осы сервисте жасай аламыз.
Екінші нұсқа B - You must keep a copy of the imported key so that you can re-import it at any time.
Идея мынада, егер біз KMS-ке импортталатын кілтпен жұмыс істейтін болсақ, қажет болған жағдайда оны бұлтқа қайта жүктеу үшін оны жергілікті жерде сақтауымыз керек.
Бұл нұсқа логикалық тұрғыдан дұрыс және оны дұрыс жауап нұсқасы ретінде белгілейміз.
Келесі нұсқа C - Keys generated in AWS KMS must be rotated every 30 days.
Негізгі KMS парақшасына қарасаңыз, C және D нұсқаларына жауап беру өте оңай.
Және бұл сервистің не туралы екендігі туралы жалпы түсінік беретін басты парақша.
Келесі - біз не үшін төлейтінімізді түсінуге арналған Pricing парақшасы.
Тағы бір маңызды парақша - Questions and answers.
Яғни, бұл сервис сұрақ-жауап форматында түсіндіріледі.
Осы парақшаларды назар аударып оқығаннан кейін сіз C нұсқасын қате деп белгілеу арқылы оңай жауап бере аласыз, себебі ол автоматты түрде әр 365 күн сайын айналады.
D нұсқасы да дұрыс емес, өйткені импортталған және ішінде жасалған кілттердің құны бірдей.
Осы видеоны жазу кезінде ол айына $1 құрайды.
Соңғы бесінші нұсқа E - You can set an expiration time for the imported key (or delete it on demand).
Бұл KMS сервисі аясындағы дұрыс мәлімдеме.
Осылайша, бұл сұраққа дұрыс жауаптар В және Е.
Осымен, біз Practice Test сұрағына талдауымызды аяқтаймыз.
Сіз жаңа ақпарат білдіңіз деп үміттенемін.
Келесі белсенділіктерімізде көріскенше!
