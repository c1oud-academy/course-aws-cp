Қайырлы күн, құрметті студенттер! 
Сіздердің барлығыңызды келесі Practice Test сұрағын талдау сессиясында көргеніме қуаныштымын.
Practice Test сұрақтары нақты AWS емтихан сұрақтарына жақын сұрақтар болып табылады.
Осылайша, біздің платформадағы Practice Test-тер бойынша жаттығып, сондай-ақ барлық Practice Test сұрақтарын талдау арқылы сіз нақты AWS емтиханын сәтті тапсыру мүмкіндігіңізді арттырасыз.
Біз талдау үшін ең қиын сұрақтарды таңдаймыз, сондықтан сіз осы сұрақтарға қалай жауап беру керектігін білгеннен кейін, қалғандарына жауап беру қиын болмайды.
Сұрақтың мазмұнына ғана емес, сонымен қатар дұрыс жауап табу үшін қандай әрекеттер жасайтынымызға назар аудару керек екенін атап өткім келеді.
Сонымен, бастайық.
Сұрақ: "Төмендегілердің қайсысы AWS жаһандық инфрақұрылымына қатысты дұрыс мәлімдеме болып табылады?"
Біз үш дұрыс жауап нұсқасын таңдауымыз керек.
Жарайды, осы сұраққа жауап беруге тырысайық.
Біз дұрыс немесе дұрыс емес екенін анық білетін сұрақтарды белгілеуіміз керек.
A нұсқасына қарайық - Edge locations exist separately to AWS regions.
Жоғарыда біз AWS аймақтары, availability аймақтары және деректер орталықтары жаһандық AWS инфрақұрылымының бөлігі болып табылатынын айттық.
Edge locations туралы айтатын болсақ, олар да жаһандық AWS инфрақұрылымының бөлігі болып табылады, бірақ CloudFront сервисіне көбірек қатысты.
Сондықтан олар кэштеу бар серверлер желісі ретінде бөлек тұрады.
Сондықтан D нұсқасы дұрыс емес.
Біз мұны анық білеміз.
Яғни, An edge location contains multiple availability zones.
Келесі нұсқа G - An AWS region contains multiple availability zones.
Бұл дұрыс мәлімдеме, сондықтан оны дұрыс деп белгілейміз.
Біз сенімді түрде атап өтуге болатын тағы бір нұсқа F - Edge locations are placed only in AWS regions.
Мұнда, D нұсқасы сияқты, Edge locations аймақтар, availability аймақтары және деректер орталықтары орналасқан жаһандық AWS инфрақұрылымына жатпайды, бірақ бөлек бар.
Сондықтан D және F жауап нұсқалары дұрыс емес.
Енді қалған үшеуінің арасынан соңғысын, үшінші дұрыс жауабын табуымыз керек.
Осы жерде мен біздің курс аясында егжей-тегжейлі талқыланбайтын, тіпті мүлдем қарастырылмайтын кейбір сервистер бар екенін атап өткім келеді.
Бұл уақытты босқа өткізіп, оқудың қажеті жоқ сұрақтар.
Дегенмен, мүмкіндігінше, кейбір сервистерді кездестіргенде, сіз бұл сервистің не үшін қолданылатыны туралы анықтаманы қарауды әдетке айналдыруыңыз керек, кем дегенде, осы сервистің негізгі парақшасымен танысыңыз.
Бұл сізге болашақта көп көмектеседі, соның ішінде нақты AWS емтиханын тапсыру, сонымен қатар сіздің көзқарасыңызды кеңейту және сізге ұсынылуы мүмкін архитектуралық шешімдердің ықтимал санын арттыру.
Қалған нұсқалар туралы айтатын болсақ, B - Wavelength, 5G-мен байланысты сервис.
C - Local Zone туралы айтатын болсақ, бұл қосымшаның әлеуетті пайдаланушысына жақындауға мүмкіндік беретін AWS аймағын кеңейтудің бір түрі.
Ал E нұсқасы - Outpost - бұл деректер орталығының, есептеу қуатының үстінен AWS-ті баптау және AWS Management Console және AWS құралдары арқылы осы жергілікті IT-ресурстарын басқару мүмкіндігі.
Осылайша, В және Е нұсқалары дұрыс емес, ал С нұсқасы дұрыс жауап нұсқасы болып табылады.
Осымен, біз Practice Test сұрағына талдауымызды аяқтаймыз.
Сіз жаңа ақпарат білдіңіз деп үміттенемін.
Келесі белсенділіктерімізде көріскенше!
