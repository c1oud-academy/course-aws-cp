Қайырлы күн, құрметті студенттер! 
Сіздердің барлығыңызды келесі Practice Test сұрағын талдау сессиясында көргеніме қуаныштымын.
Practice Test сұрақтары нақты AWS емтихан сұрақтарына жақын сұрақтар болып табылады.
Осылайша, біздің платформадағы Practice Test-тер бойынша жаттығып, сондай-ақ барлық Practice Test сұрақтарын талдау арқылы сіз нақты AWS емтиханын сәтті тапсыру мүмкіндігіңізді арттырасыз.
Біз талдау үшін ең қиын сұрақтарды таңдаймыз, сондықтан сіз осы сұрақтарға қалай жауап беру керектігін білгеннен кейін, қалғандарына жауап беру қиын болмайды.
Сұрақтың мазмұнына ғана емес, сонымен қатар дұрыс жауап табу үшін қандай әрекеттер жасайтынымызға назар аудару керек екенін атап өткім келеді.
Сонымен, бастайық.
Сұрақ: «Біз жаңа AWS аккаунтын орнаттық және жаңа IAM пайдаланушысын жасадық.
Бұл пайдаланушы өзінің AWS аккаунтына қосылуы үшін оған қандай sign-in URL беруіміз керек?»
Біз бір дұрыс жауапты таңдауымыз керек.
Бұл сұрақ бірегей және дұрыс жауап беру үшін біз пайдаланып жатқан sign-in URL құрылымын білуіңіз керек.
Әдетте, бұлтпен жиі жұмыс істегенде, сіз бұл URL мекенжайын жаттап аласыз және көрген мысалдарды еске түсіріп, бұл сұраққа дұрыс жауап бере аласыз.
Біздің жағдайда консольмен жұмыс істеу тәжірибесі соншалықты көп емес, бірақ соған қарамастан біз Lab белсенділіктерін өтіп жатырмыз.
Sign-in URL мекенжайларын көріп, олармен жұмыс істегенімізді есіңізде болса, сіз дұрыс жауап бере аласыз.
Барлық басқа жағдайларда сіз дұрыс жауапты білуіңіз керек.
Консольге өту қажет болғанда /accountId арқылы кіру мүлдем қисынсыз, біз логикалық тұрғыдан B нұсқасын алып тастаймыз.
Ал A, C және D нұсқалары сәйкес келеді, өйткені оларда account ID немесе alias бар, sign-in бөлігі бар, ол тек А нұсқасында жоқ, біз оны осылай алып тастай аламыз және олардың барлығы console деп аяқталады.
Логикалық түрде үш нұсқа да сәйкес.
Дұрыс жауап С нұсқасы.
Sign-in URL мекенжайы account ID немесе alias, одан кейін sign-in, кейін aws.amazon.com бұл бір тұтас бөлік, кейін /console деп басталатынын есте ұстаған жөн
Есте сақтауды қажет ететін ерекше сәттер көп емес, бірақ олар маңызды.
Бұл ақпарат AWS ішінде жұмыс істегенде үнемі пайдаланылады, сондықтан сіз оны есте сақтап алғаныңыз жөн болады.
Мен дәрістерімізде, соның ішінде Practice Test сұрақтарын талдауда осы сұрақтардың барлығын атап көрсететін боламын.
Осымен, біз Practice Test сұрағына талдауымызды аяқтаймыз.
Сіз жаңа ақпарат білдіңіз деп үміттенемін.
Келесі белсенділіктерімізде көріскенше!
