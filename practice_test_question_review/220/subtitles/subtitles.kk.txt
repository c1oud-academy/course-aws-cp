Қайырлы күн, құрметті студенттер! 
Сіздердің барлығыңызды келесі Practice Test сұрағын талдау сессиясында көргеніме қуаныштымын.
Practice Test сұрақтары нақты AWS емтихан сұрақтарына жақын сұрақтар болып табылады.
Осылайша, біздің платформадағы Practice Test-тер бойынша жаттығып, сондай-ақ барлық Practice Test сұрақтарын талдау арқылы сіз нақты AWS емтиханын сәтті тапсыру мүмкіндігіңізді арттырасыз.
Біз талдау үшін ең қиын сұрақтарды таңдаймыз, сондықтан сіз осы сұрақтарға қалай жауап беру керектігін білгеннен кейін, қалғандарына жауап беру қиын болмайды.
Сұрақтың мазмұнына ғана емес, сонымен қатар дұрыс жауап табу үшін қандай әрекеттер жасайтынымызға назар аудару керек екенін атап өткім келеді.
Сонымен, бастайық.
Сұрақ: «Қандай AWS сервистері аймақтық болып табылады?»
Бұл сұраққа үш дұрыс жауапты таңдау керек.
Алдын ала дұрыс немесе дұрыс емес жауап нұсқаларын алып тастап, осы сұраққа жауап беруге тырысайық.
B және G нұсқаларына қарап көрейік, Amazon CloudFront, сондай-ақ AWS IAM сервистері ұсынылатынын көреміз.
Бұл көптеген аймақтарда жаһандық деңгейде жұмыс істейтін сервистер.
Сондықтан бұл нұсқалар дұрыс емес.
Егер біз A және C нұсқалары туралы айтатын болсақ, бұл EC2 Instances және EBS Volumes, онда бізде аймақтық деңгейде жұмыс істейтін VPC бар екенін есте ұстайық.
Төменде VPC ішінде availability zone деңгейінде жұмыс істейтін subnets бар.
Яғни, біз бірнеше топтарда subnet құра алмаймыз.
Және бір subnet тек бір нақты availability zone-да бола болады.
Subnet-те осы EC2 инстанстарына байланыстырылған EC2 Instances бен EBS Volumes бар.
Осылайша, subnets availability zone деңгейінде жұмыс істейтін ресурстар болғандықтан, EC2 Instances және EBS Volumes аймақ деңгейінде жұмыс істемейді.
Сондықтан А және С нұсқаларын алып тастауға болады.
Осылайша, бізде Auto Scaling Groups, Security Groups, сондай-ақ аймақ деңгейінде жұмыс істейтін Elastic Load Balancer нұсқалары қалды, яғни бірнеше availability zone үстінде.
Осымен, біз Practice Test сұрағына талдауымызды аяқтаймыз.
Сіз жаңа ақпарат білдіңіз деп үміттенемін.
Келесі белсенділіктерімізде көріскенше!
