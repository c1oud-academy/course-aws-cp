Қайырлы күн, құрметті студенттер! 
Сіздердің барлығыңызды келесі Practice Test сұрағын талдау сессиясында көргеніме қуаныштымын.
Practice Test сұрақтары нақты AWS емтихан сұрақтарына жақын сұрақтар болып табылады.
Осылайша, біздің платформадағы Practice Test-тер бойынша жаттығып, сондай-ақ барлық Practice Test сұрақтарын талдау арқылы сіз нақты AWS емтиханын сәтті тапсыру мүмкіндігіңізді арттырасыз.
Біз талдау үшін ең қиын сұрақтарды таңдаймыз, сондықтан сіз осы сұрақтарға қалай жауап беру керектігін білгеннен кейін, қалғандарына жауап беру қиын болмайды.
Сұрақтың мазмұнына ғана емес, сонымен қатар дұрыс жауап табу үшін қандай әрекеттер жасайтынымызға назар аудару керек екенін атап өткім келеді.
Сонымен, бастайық.
Сұрақ: «Қандай VPC endpoint пен сыртқы IP-адрестер Amazon S3 bucket-ке сұраныс жасай алатынын қалай таңдауға болады?»
Бұл сұрақта сіз бір дұрыс жауапты таңдауыңыз керек.
Жарайды, сұраққа жауап беруге тырысайық және анық дұрыс емес жауап нұсқаларын алып тастаймыз.
A нұсқасына қарайық - Set up a Site-to-Site VPN connection between the VPC or External IP and the S3 bucket.
Сұрақ бізде жергілікті деректер орталығы, жергілікті инфрақұрылым бар екендігі туралы ештеңе айтпайды және осыған байланысты Site-to-Site VPN туралы айтудың қажеті жоқ, бұл нұсқа сәйкес емес.
C нұсқасы туралы да соны айтуға болады - Direct connect, ол да сол мақсаттарда, яғни жергілікті on-premise дата-центрін бұлтпен қосу үшін қолданылады.
Сұрақта бұл туралы нақты айтпағандықтан, тек VPC endpoint бұлттың ішіндегі барлық нәрсе екенін айтқандықтан, біз C нұсқасын да алып тастаймыз.
B нұсқасын қарайық - Use an IAM policy to specify which VPC endpoints or external IP addresses can access the S3 bucket.
Жақсы жауап.
Біз IAM policy жасадық, енді бұл policy-ді қай identity-мен байланыстыру керек?
Пайдаланушыға?
Рольге?
Әлде топқа ма?
Біріншісі де, екіншісі де, үшінші нұсқасы да жарамайды, өйткені біздің жағдайда ресурс - bucket.
Сондықтан біз В нұсқасын 50/50 деп белгілей аламыз, оған күмәнданамыз.
Әрі қарай, D нұсқасын қарастырайық - Use a bucket policy to specify which VPC endpoints or external IP addresses can access the S3 bucket.
Бұл да policy, бірақ бұл жағдайда біз нақты bucket үшін policy жасайтын жергілікті S3 bucket-тің функциясын қолданамыз.
Біз бұл нұсқа сәйкес және бұл сұраққа дұрыс жауап екенін көреміз.
Осымен, біз Practice Test сұрағына талдауымызды аяқтаймыз.
Сіз жаңа ақпарат білдіңіз деп үміттенемін.
