Қайырлы күн, құрметті студенттер! 
Сіздердің барлығыңызды келесі Practice Test сұрағын талдау сессиясында көргеніме қуаныштымын.
Practice Test сұрақтары нақты AWS емтихан сұрақтарына жақын сұрақтар болып табылады.
Осылайша, біздің платформадағы Practice Test-тер бойынша жаттығып, сондай-ақ барлық Practice Test сұрақтарын талдау арқылы сіз нақты AWS емтиханын сәтті тапсыру мүмкіндігіңізді арттырасыз.
Біз талдау үшін ең қиын сұрақтарды таңдаймыз, сондықтан сіз осы сұрақтарға қалай жауап беру керектігін білгеннен кейін, қалғандарына жауап беру қиын болмайды.
Сұрақтың мазмұнына ғана емес, сонымен қатар дұрыс жауап табу үшін қандай әрекеттер жасайтынымызға назар аудару керек екенін атап өткім келеді.
Сонымен, бастайық.
Сұрақ: "IAM пайдаланушысы соңғы рет AWS жүйесіне консоль арқылы немесе бағдарлама арқылы кіргенін қайдан тексере аламыз?"
Біз бір дұрыс жауапты таңдауымыз керек.
Бұл сұраққа анық дұрыс емес жауап нұсқаларын алып тастау арқылы жауап беруге тырысайық.
Бізге IAM пайдаланушысы туралы ақпарат қажет екендігі айтылды.
Бұдан шығатыны, бізге AWS IAM сервисі қажет.
Егер біз A және B нұсқалары туралы айтатын болсақ, онда IAM сервисі туралы ештеңе айтылмайды, бірақ негізгі AWS Console парақшасында кейбір жаһандық параметрлерді ашу ұсынылады, бұл мүлдем қисынды емес, сондықтан A және B нұсқаларын алып тастауға болады.
Әрі қарай жақындаймыз.
Бізден IAM Console ашуды сұрайды, яғни AWS Management Console ішінде AWS IAM сервисін және сол жерде C нұсқасын - Users таңдауыңыз керек және ішінде әрбір пайдаланушы үшін Last Activity көре аласыз.
D нұсқасы бұл ақпаратты көру үшін Credential Report есебін басып, содан кейін осы есепті жүктеп алу керек екенін айтады.
Бұл жерде мен AWS Management Console білуді талап ететін сұрақтар өте сирек кездесетінін атап өткім келеді.
Нақты AWS емтиханында бір, ең көбі екі сұрақ кездесер.
Сондықтан, Management Console-ді егжей-тегжейлі зерделеуге уақытты жоғалтпаңыз, керісінше, осы сервистерді, олардың не үшін қажет екенін, бір-бірімен қалай жұмыс істей алатынын және т.б. түсінуге назар аударыңыз.
Мұндай сұрақтар туындаған кезде сіз мүмкіндігінше дұрыс емес сұрақтарды алып тастауға тырысыңыз.
Жауаптың бірнеше нұсқасы қалса, сол немесе басқа жауап нұсқасын қандай да логикалық тұрғыдан негіздеуге тырысыңыз.
Егер олай болмаса, кездейсоқ жауап беріңіз.
Бұл сұрақтан ұпай жоғалтамын деп мүлдем алаңдамаңыз, өйткені өту балы өте кең, яғни бұл емтиханды тапсыру үшін плюс-минус 70% қажет.
Бұл емтихан оңай дегенді білдірмейді; көптеген сұрақтарға дұрыс жауап беру керек.
Мұндағы ой мынада: қай нұсқаның дұрыс екенін ойлап уақытты жоғалтып жатқанда немесе күмәнданып отырып, емтихан аясында қымбат уақытыңызды жоғалтқанда, сіз оңайырақ немесе басқа әртүрлі сұрақтарға дұрыс жауап бере алмай қаласыз.
Осы сияқты сұрақтардағы идея мынада: сіз бұл сұрақты белгілейсіз, егер қаласаңыз, оған жауап бересіз, егер жауап бере алмасаңыз, белгілеп, әрі қарай басқа сұрақтарға өтесіз.
Барлық сұрақтарды бір рет өткеннен кейін сіз барлық белгіленген сұрақтарды қосымша қарап шығып, оларға қандай да бір жолмен жауап беруге тырысасыз.
Бұл сұрақтың жауабы ретінде D нұсқасы дұрыс емес, ал С нұсқасы дұрыс.
Логикалық тұрғыдан мұны келесідей түсіндіруге болады.
Last Activity туралы ақпарат өте маңызды ақпарат болып табылады және оны IAM пайдаланушыларымен жасау, жою немесе басқа әрекеттермен жұмыс істейтін әкімші үшін көру жақсы болар еді.
Сондықтан, ол AWS Management Console-інде болуы мүмкін және қосымша әрекеттерді қажет ететін кез келген есепті қосымша жүктеп алудың қажеті жоқ.
Осыған сүйене отырып, біз C нұсқасын таңдау арқылы бұл сұраққа логикалық тұрғыдан жауап береміз.
Осымен, біз Practice Test сұрағына талдауымызды аяқтаймыз.
Сіз жаңа ақпарат білдіңіз деп үміттенемін.
Келесі белсенділіктерімізде көріскенше!
