Қайырлы күн, құрметті студенттер! 
Сіздердің барлығыңызды келесі Practice Test сұрағын талдау сессиясында көргеніме қуаныштымын.
Practice Test сұрақтары нақты AWS емтихан сұрақтарына жақын сұрақтар болып табылады.
Осылайша, біздің платформадағы Practice Test-тер бойынша жаттығып, сондай-ақ барлық Practice Test сұрақтарын талдау арқылы сіз нақты AWS емтиханын сәтті тапсыру мүмкіндігіңізді арттырасыз.
Біз талдау үшін ең қиын сұрақтарды таңдаймыз, сондықтан сіз осы сұрақтарға қалай жауап беру керектігін білгеннен кейін, қалғандарына жауап беру қиын болмайды.
Сұрақтың мазмұнына ғана емес, сонымен қатар дұрыс жауап табу үшін қандай әрекеттер жасайтынымызға назар аудару керек екенін атап өткім келеді.
Сонымен, бастайық.
Сұрақ: «Bastion Host дегеніміз не?»
Сіз бір дұрыс жауапты таңдауыңыз керек.
Жарайды, қате жауап нұсқаларын алып тастап, осы сұраққа жауап беруге тырысайық.
Бізде ұзын жауап нұсқалары бар екенін көреміз.
Тағы да, алаңдамаңыз; керісінше, бұл кейбір нұсқаларды оңай алып тастауға көмектеседі.
Bastion Host деген не екенін қарапайым сөзбен еске салайын.
Бұл 22 порт арқылы, яғни жергілікті кеңседен немесе жергілікті инфрақұрылымнан SSH арқылы қосылу үшін VPC-дің public subnet-інде орналастыратын инстанс.
Одан әрі, осы серверге қосылғаннан кейін, бұл серверде сәйкес Network ACL және Security топтарын тіркей отырып, private subnet-те орналасқан ресурстарға, инстанстарға немесе дерекқорларға қосылу мүмкіндігі бар.
Осылайша, жергілікті түрде кеңсеңізден private subnet-ке және сол жерде орналасқан және Интернеттен тікелей қол жетімді емес барлық ресурстарға қосыла аласыз.
Bastion Host-ты басқаша Jump Server деп атайды, яғни алдымен осы Bastion хостына "секіресіз", содан кейін private subnet-ке өтесіз.
Жауап нұсқаларын қарастырайық.
В нұсқасына қараңыз.
Әрине, сіз нұсқаны толығымен оқып шығуыңыз керек, бірақ басында біз Bastion Host is a virtual network card деп тұрғанын көреміз.
Bastion Host желілік карта емес, сондықтан бұл нұсқа дұрыс емес екені анық.
D нұсқасына қарайық - A Bastion Host is a special locked-down environment for hosting an RDS Database.
Яғни, Bastion Host белгілі бір орта болып табылады деп көрсетілген.
Бұл да дұрыс емес, біз оны алып тастаймыз.
C нұсқасына қарайық - A Bastion Host is a special EC2 instance used to connect to AWS Direct Connect.
Иә, бұл нұсқаның айласы бар, яғни бірінші бөлігі дұрыс.
Яғни, иә, бұл белгілі бір мақсатқа арналған арнайы EC2 инстанс.
Соңғы A нұсқасына қарайық - A Bastion Host is an intermediary instance that provides access to a private network from an external network.
Яғни, Bastion Host-қа қатысты ең дұрыс мәлімдеме - А нұсқасы.
С нұсқасының бірінші бөлігі дұрыс болғанымен, екіншісіне күмәндануға болады.
А нұсқасы толығымен дұрыс болғандықтан, оның ішіндегі барлық мәлімдемелер ақиқат, сондықтан бұл нұсқа дұрыс деп жауап бере аламыз.
Осымен, біз Practice Test сұрағына талдауымызды аяқтаймыз.
Сіз жаңа ақпарат білдіңіз деп үміттенемін.
