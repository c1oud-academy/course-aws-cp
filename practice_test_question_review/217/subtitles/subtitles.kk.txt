Қайырлы күн, құрметті студенттер! 
Сіздердің барлығыңызды келесі Practice Test сұрағын талдау сессиясында көргеніме қуаныштымын.
Practice Test сұрақтары нақты AWS емтихан сұрақтарына жақын сұрақтар болып табылады.
Осылайша, біздің платформадағы Practice Test-тер бойынша жаттығып, сондай-ақ барлық Practice Test сұрақтарын талдау арқылы сіз нақты AWS емтиханын сәтті тапсыру мүмкіндігіңізді арттырасыз.
Біз талдау үшін ең қиын сұрақтарды таңдаймыз, сондықтан сіз осы сұрақтарға қалай жауап беру керектігін білгеннен кейін, қалғандарына жауап беру қиын болмайды.
Сұрақтың мазмұнына ғана емес, сонымен қатар дұрыс жауап табу үшін қандай әрекеттер жасайтынымызға назар аудару керек екенін атап өткім келеді.
Сонымен, бастайық.
Сұрақ: “Төмендегілердің қайсысы жаһандық AWS сервистері болып табылады?”
Сіз үш дұрыс нұсқаны таңдауыңыз керек.
Бұл сұраққа жауап беру үшін біз сенімді немесе сенімді емес нұсқаларды алып тастауға тырысайық.
C нұсқасын қараңыз - “Amazon VPC”.
Біздің есімізде, Amazon VPC бір аймақ аясында құрылған және бірнеше availability zone-да болуы мүмкін.
Егер біз бірнеше аймақпен жұмыс жасайтын болсақ, онда біз әр аймақ үшін жеке VPC құруымыз керек.
Бұл Amazon VPC жаһандық емес екенін білдіреді, сондықтан бұл дұрыс жауап емес.
Біздің VPC-де subnet бар, subnet ішінде EC2 инстанстарымыз бар және инстанстарымызға EBS Volume бекітілген.
Бұл D нұсқасы.
VPC жаһандық сервис болмағандықтан, EBS Volume да AWS жаһандық сервисі емес.
F нұсқасына қараңыз - AWS IAM.
Бұл жаһандық сервис болып табылады және кез келген аймаққа байланысты емес.
Егер сіз пайдаланушыны жасасаңыз, мысалы, IAM user, онда сол пайдаланушы қол жетімді болады және AWS аккаунты аясында кез келген AWS аймағында тиісті құқықтар берілсе, жұмыс істей алады.
Келесі екі нұсқа - B және G, біз оларды бірінші рет көріп отырмыз деп елестетіп көрейік, сондықтан біз оларды сары түспен 50/50 белгілейміз.
Біз бұған күмәнданамыз.
A нұсқасын қарастырайық - Amazon Route 53.
Бұл Интернет-трафикті қажетті IT-ресурстарға бағыттауға мүмкіндік беретін сервис.
Бізде трафикті бірнеше аймаққа бағыттай алатын routing policy нұсқасы бар екенін есте сақтаймыз.
Бір аймақта проблемалар туындаған жағдайда, трафик екінші резервтік аймаққа бағытталады.
Сонымен, бұл қазірдің өзінде Amazon Route 53 жаһандық AWS сервисі екенін білдіреді.
Amazon CloudFront та өте айқын сервис.
Неге? Себебі CloudFront - CDN болып табылады.
CDN - бұл Content Delivery Network, яғни бұл сіздің аймағыңызды таратуға және бүкіл әлем бойынша барлық AWS сервистерінде қол жетімді болуға мүмкіндік беретін сервис.
Осылайша, Amazon CloudFront бұл сұраққа дұрыс жауап болып табылады.
Осымен Practice Test сұрағын талдауды аяқтаймыз.
Сіз жаңа ақпарат білдіңіз деп үміттенемін.
Келесі белсенділіктерімізде көріскенше!
