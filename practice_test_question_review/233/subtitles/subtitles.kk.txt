Қайырлы күн, құрметті студенттер! 
Сіздердің барлығыңызды келесі Practice Test сұрағын талдау сессиясында көргеніме қуаныштымын.
Practice Test сұрақтары нақты AWS емтихан сұрақтарына жақын сұрақтар болып табылады.
Осылайша, біздің платформадағы Practice Test-тер бойынша жаттығып, сондай-ақ барлық Practice Test сұрақтарын талдау арқылы сіз нақты AWS емтиханын сәтті тапсыру мүмкіндігіңізді арттырасыз.
Біз талдау үшін ең қиын сұрақтарды таңдаймыз, сондықтан сіз осы сұрақтарға қалай жауап беру керектігін білгеннен кейін, қалғандарына жауап беру қиын болмайды.
Сұрақтың мазмұнына ғана емес, сонымен қатар дұрыс жауап табу үшін қандай әрекеттер жасайтынымызға назар аудару керек екенін атап өткім келеді.
Сонымен, бастайық.
Сұрақ: "AWS VPC ішіндегі AWS Management Console-де төмендегілердің қайсысын басқара аласыз?"
Сізге үш дұрыс жауап нұсқасын таңдау керек.
Бұл жерде мен AWS Management Console-де осы немесе басқа сервистің қай жерде орналасқанын білуге арналған осындай сұрақтар, егер олар нақты AWS емтиханында болса да, өте сирек кездесетінін атап өткім келеді.
Сондықтан, сіз осындай сұрақтарды, сондай-ақ AWS Management Console-ін егжей-тегжейлі зерттеуге уақытыңызды жоғалтпауыңыз керек.
Бұл сұрақтарға логикалық тұрғыдан жауап беру керек, және осы сұрақта солай істеп көрейік.
VPC орнатқан кезде немен жұмыс істегенімізді еске түсірейік.
Бұл тізімге қарасақ, NAT Gateways мен Security Groups айқын үміткерлер болып табылады.
Сондықтан, біз бұл нұсқаларды дұрыс деп белгілей аламыз, өйткені Security Groups пен Network ACLs, соның ішінде NAT Gateways VPC құру кезінде негізгі құрамдастардың бірі болып табылады.
Басқа нұсқаларды қарастыратын болсақ, бұл Launch Templates және EBS Volumes, олар көбінесе EC2-ге қатысты.
Яғни, виртуалды серверді орнатқан кезде біз бірнеше қадамдардан өтуіміз керек, соның ішінде Launch Templates және EBS Volumes таңдау.
Сондықтан біз бұл жауап нұсқаларын алып тастай аламыз.
Жауаптың екі нұсқасы қалды, олар C - Egress Only Internet Gateways.
Атауына сүйене отырып, біз білмейміз деп ойласақ та, бұл Интернет-трафик пен желілерге қатысты нәрсе деп қорытынды жасауға болады.
Егер біз F опциясы туралы айтатын болсақ, бұл Capacity Reservations.
VPC сервисіне және желілерге қатысты емес.
Сондықтан сіз жауапты білмесеңіз де, логикалық тұрғыдан F нұсқасын алып тастап, C нұсқасын дұрыс деп жауап беруге болады.
Шын мәнінде, бұл солай.
Бұл сұраққа A, C және E үш нұсқасы дұрыс.
Осымен, біз Practice Test сұрағына талдауымызды аяқтаймыз.
Сіз жаңа ақпарат білдіңіз деп үміттенемін.
Келесі белсенділіктерімізде көріскенше!
